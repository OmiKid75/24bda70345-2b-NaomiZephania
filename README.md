## Name

## Objective
To design and develop a modern, responsive product filtering web application using Next.js and TypeScript, allowing users to easily browse and filter products based on selected criteria.

## Technologies Used
- Next.js
- React
- TypeScript
- Tailwind CSS
- Github(Version Control)
- Vercel (Deployment)

## Features
- Dynamic product filtering
- Clean and modern user interface
- Fully responsive design
- Reusable UI components
- Type-safe development using TypeScript
- Optimized production build

## Folder Structure
- app/ → Main application routes and pages
- components/ → Reusable UI components
- lib/ → Helper functions and utilities
- types/ → Shared TypeScript type definitions
- public/ → Static assets
- README.md

## How to Run Locally
1. Download or clone the repository
2. Open index.html in a web browser

## Deployment
This project is deployed on Vercel using GitHub integration.
